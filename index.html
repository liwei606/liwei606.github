Projects

MapReduce-based PageRank

  • A Java implementation of the PageRank algorithm based on the MapReduce model.
  • It was deployed and evaluated on a three-node Hadoop cluster.
  • Three optimization approaches were employed to reduce the time cost of both network transmission and disk accesses.

SmallC Compiler

  • A compiler written in C, which targets the MIPS architecture.
  • It consists of four phases including syntactic analysis using flex and bison, semantic analysis and type checking, code generation, and peephole optimization.

C2Java Translator

  • A translator implemented on the abstract syntax tree (AST) level, where the AST is first generated from C source codes and then translated into Java.
